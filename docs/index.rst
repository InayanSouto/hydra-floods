HYDrologic Remote sensing Analysis for Floods (HYDRAFloods)
===========================================================


HYDRAFloods is an open source Python application for downloading, processing, and delivering flood maps derived from remote sensing data.
The bases behind the tool is to use as many remote sensing dataset as possible to provide daily (sometime twice daily) flood maps.
